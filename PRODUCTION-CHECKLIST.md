# Weekly Production Checklist

Target: ≤ 90 minutes total, Thursday afternoon, ships Friday 7:00 AM local.

## 1. Curate (Mon–Thu, ~5 min/day)

- [ ] Scan sources; save candidate links to a running note (Notion, Apple Notes, whatever — just one place)
- [ ] Aim for 12-15 candidates; you'll keep 10-12
- Sources: The Verge / TechCrunch AI, ZDNet, Microsoft 365 blog, OpenAI blog, Anthropic blog, TLDR AI, LinkedIn

## 2. Draft (Thu, ~40 min)

- [ ] Copy `templates/issue-template.md` → `issues/issue-NNN.md`
- [ ] Fill each section; cut anything that needs jargon to explain
- [ ] Write subject line **last** — it should be the single best item in the issue
- [ ] Preview text ≠ first line of the email (write it deliberately; it's your second subject line)

## 3. Fact-check gate (Thu, ~15 min) — nothing ships without this

- [ ] **Every stat has a live link** to a primary source (the survey/report itself, not a blog citing it)
- [ ] **Every headline link works** and goes where the headline promises
- [ ] **Product claims verified this week** — AI tools change monthly. Feature availability, pricing, and *especially privacy/data-training policies* go stale fast. If you last verified a claim more than ~30 days ago, re-verify.
- [ ] **No empty links.** `[]()` placeholders are for drafting only
- [ ] Read-time estimates present on every linked item

## 4. Compliance (one-time setup, then automatic)

TRLeadership is Canadian → **CASL applies** (stricter than US CAN-SPAM):

- [ ] Express consent recorded for every subscriber (date + method) — your email platform should do this; confirm double opt-in is enabled
- [ ] Sender clearly identified (business name + mailing address in footer)
- [ ] Working unsubscribe in every send, honored within 10 days
- [ ] Don't import the existing TRLeadership list wholesale unless consent for *this newsletter* is defensible — safest path: email the existing list once inviting them to opt in

## 5. Send (Fri)

- [ ] Send a test to yourself; check on phone (60%+ of opens are mobile)
- [ ] Check dark mode rendering
- [ ] Schedule for 7:00 AM Friday
- [ ] Post one item as a LinkedIn teaser Friday mid-morning, linking to the web version

## 6. Review (5 min, following Mon)

- [ ] Log open rate, click rate, top-clicked item in a simple spreadsheet
- [ ] The top-clicked section tells you what to lead with next week

---

## Kill criteria (decide up front, revisit at issue 12)

- Open rate < 30% for 4 consecutive issues → rework subject lines and lead section
- List growth < 5%/month after issue 8 → rework the CTA/growth loop before writing more issues
- If you dread Thursday afternoons by issue 6 → cut a section. Four sections shipped weekly beats five shipped sporadically.
